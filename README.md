# Павлов Вадим Алексеевич — IT архитектор возможностей

Персональный сайт и портфолио. Фокус на архитектуре цифровых продуктов, стратегиях развития и практичных решениях.

## Технологии

- Vite
- React
- TypeScript
- Framer Motion
- Tailwind CSS

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
